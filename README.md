# ✈️ Flight Booking System

A modern **Flight Booking System** developed using **ASP.NET Framework (Web Forms)** and **C#**. This project allows users to search for flights, book tickets, manage reservations, and receive notifications through a user-friendly web interface. It also includes a comprehensive admin panel for managing flights, users, bookings, and system content.

---

## 📌 Features

### 👤 User Features

* User Registration & Login
* Email OTP Verification
* Flight Search
* Flight Booking
* View Booked Tickets
* Cancel Bookings
* User Profile Management
* Forgot Password
* Notifications
* Booking Confirmation
* Payment Success Page
* Responsive User Interface

### 👨‍💼 Admin Features

* Admin Dashboard
* Manage Flights
* Manage Flight Status
* Manage Users
* Manage Payments
* Manage Discounts
* Manage Hot Deals
* Manage Notifications
* Travel News Management
* Flight Statistics
* User Activity Logs
* Customer Support Management

## 🚀 Getting Started

### Prerequisites

* Visual Studio 2022
* .NET Framework
* SQL Server
* IIS Express
* Git

### Installation

1. Clone the repository.

```bash
git clone https://github.com/Dhvanit2023/flightbooking.git
```

2. Open the solution in **Visual Studio 2022**.

3. Restore all NuGet packages.

4. Create the SQL Server database.

5. Update the database connection string in **Web.config**.

6. Configure your Email SMTP settings.

7. (Optional) Configure Twilio credentials if SMS OTP is required.

8. Build and run the project.
